# Índice de Ejercicios Resueltos (EDO)

Esta carpeta contiene pautas de ejercicios resueltos que sirven como **ejemplo de estilo y formato** (Few-Shot Prompting). Al resolver un ejercicio, el LLM debe buscar el ejemplo más similar en esta carpeta y copiar estrictamente su estilo directo, conciso y de "estudiante rindiendo certamen".

## 01. Primer Orden (`01_primer_orden/`)
- `01_separable.md` - Ecuaciones de variables separables.
- `02_lineal.md` - Ecuaciones lineales (Factor Integrante).
- `03_exacta.md` - Ecuaciones exactas.
- `04_bernoulli.md` - Ecuación de Bernoulli.

## 02. Segundo Orden (`02_segundo_orden/`)
- `01_homogenea_constante.md` - Homogénea con coeficientes constantes.
- `02_coeficientes_indeterminados.md` - No homogénea (Coeficientes Indeterminados).
- `03_variacion_parametros.md` - No homogénea (Variación de Parámetros).
- `04_cauchy_euler_no_homogenea.md` - Ecuación de Cauchy-Euler no homogénea con límite integral.

## 03. Transformada de Laplace (`03_laplace/`)

- `02_inversa_fracciones_parciales.md` - Transformada Inversa.

## 04. Teoría y Problemas de Frontera (`04_teoria_y_frontera/`)

## 05. Sistemas de EDOs (`05_sistemas/`)
- `01_sistema_diagonalizacion.md` - Sistemas de segundo orden de la forma $\mathbf{x}'' = A^2\mathbf{x}$ usando diagonalización de matrices.
