\section*{Objetivos, Metodología y Resultados Esperados}

\subsection*{Objetivo General}
Implementar un sistema de software clínico que integre inteligencia artificial para el análisis
automatizado de registros médicos y la predicción de eventos clínicos en pacientes de Unidades
de Cuidados Intensivos (UCI).

\subsection*{Metodología y Objetivos Específicos}
El trabajo se divide en tres áreas de desarrollo consecutivas:

\begin{itemize}
    \item \textbf{Área 1: Software de Extracción (Mayo – Junio)}\\
    Desarrollo de una herramienta de anotación clínica para generar el \textit{ground truth},
    respaldada por una base de datos consultable y expuesta mediante una API REST (FastAPI).
    Se implementará un front-end de evaluación para testear modelos LLM open-source.

    \item \textbf{Área 2: Análisis Predictivo con Machine Learning (Junio – Julio)}\\
    Generación de análisis predictivo sobre los campos extraídos para predecir
    \textit{outcomes} clínicos (ej.\ mortalidad, shock séptico). Se compararán diversos
    algoritmos de clasificación tradicionales versus extracción semántica, aplicando técnicas
    de interpretabilidad clínica (valores SHAP).

    \item \textbf{Área 3: Modelo Transformer para Predicción (Julio – Septiembre)}\\
    Estudio de arquitecturas orientadas a predicción de eventos clínicos (Delphi 2M, TFT,
    Foresight). Se realizará la adaptación y \textit{fine-tuning} del modelo sobre datos
    reales de la UCI, finalizando con una validación rigurosa mediante métricas de desempeño
    (AUROC, precisión, recall, F1-score).
\end{itemize}

\subsection*{Resultados Esperados}
Un sistema integral funcional que permita la extracción automatizada de texto clínico, análisis
predictivo interpretable y modelos avanzados de inteligencia artificial ajustados a las
necesidades reales de gestión de pacientes críticos en la UCI.

\section*{Alineamiento con el Perfil de Egreso}

El proyecto está diseñado para evidenciar las siguientes competencias técnicas del perfil de
Ingeniería Civil de Industrias y Computación:

\begin{enumerate}
    \item Desarrollar soluciones innovadoras basadas en conocimientos avanzados de Ingeniería de
    Computación.
    \item Aplicar diversos métodos de análisis de datos para la comprensión de los fenómenos abordados.
    \item Investigar sobre nuevas tecnologías de información existentes en la industria y facilitar
    su adopción dentro de las organizaciones.
\end{enumerate}

\subsection*{Evidencias de Ejecución}
\begin{itemize}
    \item \textbf{Evidencia ``Desarrollar soluciones innovadoras basadas en conocimientos avanzados de Ingeniería de Computación'' (Área 1):} Diseño, arquitectura e implementación
    integral de la herramienta de software clínico (API REST, base de datos e interfaz),
    adaptada a la problemática y requerimientos reales de la Unidad de Paciente Crítico.

    \item \textbf{Evidencia ``Aplicar diversos métodos de análisis de datos para la comprensión de los fenómenos abordados'' (Área 2):} Desarrollo del modelo predictivo de
    \textit{Machine Learning} para la predicción de eventos clínicos, evaluando comparativamente
    algoritmos y aplicando métodos de interpretabilidad (valores SHAP) sobre datos médicos.

    \item \textbf{Evidencia ``Investigar sobre nuevas tecnologías de información existentes en la industria y facilitar su adopción dentro de las organizaciones'' (Área 3):} Estudio profundo, adaptación y ejecución
    del proceso de \textit{fine-tuning} sobre arquitecturas Transformer especializadas
    (Delphi 2M, TFT, Foresight), facilitando la adopción de estas nuevas tecnologías en la UCI.
\end{itemize}
