# Perfil Profesional Ingeniero Civil de Industrias
**Diploma en Ingeniería de Computación - Programa de Estudios 2013**

Para el título de Ingeniero Civil de Industrias con Diploma en Ingeniería de Computación, se definen los siguientes objetivos educacionales y competencias de egreso.

---

## Objetivos Educacionales y Competencias

### 1. Excelencia en Ingeniería de Computación
*Desempeñarse de forma ejemplar en los ámbitos de la Ingeniería de Computación, sobre la base de un conocimiento avanzado en Ciencia de la Computación, Ingeniería de Software y Sistemas de Información.*

#### **a. Aplicación de Conocimientos Avanzados**
*Aplicar conocimientos avanzados de Ciencia de la Computación, Ingeniería de Software y Sistemas de Información para entender problemas complejos y abiertos.*
- IIC1253 - Matemáticas discretas
- IIC2133 - Estructuras de datos y algoritmos
- IIC2143 - Ingeniería de software
- IIC2154 - Proyecto de especialidad
- IIC2233 - Programación avanzada
- IIC2713 - Sistemas de información

#### **b. Desarrollo de Soluciones Innovadoras**
*Desarrollar soluciones innovadoras basadas en conocimientos avanzados de Ingeniería de Computación.*
- IIC2343 - Arquitectura de computadores
- IIC2143 - Ingeniería de software
- IIC2413 - Bases de datos
- IIC2333 - Sistemas operativos y redes
- IIC2154 - Proyecto de especialidad

#### **c. Calidad de Software**
*Analizar las necesidades de calidad de un modelo o artefacto de software, diseñar soluciones que implementen las estrategias de calidad determinadas, implementar y evaluar el impacto de las decisiones de diseño.*
- IIC2113 - Diseño detallado de software
- IIC2143 - Ingeniería de software
- IIC2173 - Arquitectura de sistemas de software
- IIC2413 - Bases de datos
- IIC3143 - Desarrollo de software
- IIC3745 - Testing

---

### 2. Innovación y Responsabilidad Social
*Innovar en el diseño de modelos, productos y/o sistemas que generen soluciones a necesidades de la ingeniería de computación, considerando criterios de responsabilidad profesional, ética y social.*

#### **a. Diseño y Desarrollo Ético**
*Diseñar y desarrollar modelos y artefactos de software y simular soluciones a problemas de la Ciencia e Ingeniería de Computación, cumpliendo con restricciones técnicas, sociales y éticas.*
- IIC2113 - Diseño detallado de software
- IIC2143 - Ingeniería de software
- IIC2154 - Proyecto de especialidad
- IIC2173 - Arquitectura de sistemas de software
- IIC2233 - Programación avanzada
- IIC2413 - Bases de datos
- IIC2513 - Tecnologías y aplicaciones web

#### **b. Análisis Sistemático de Problemáticas**
*Analizar en forma sistemática las diferentes problemáticas de los usuarios, y diseñar productos o sistemas de software que queden expresados mediante algún lenguaje de programación, de acuerdo a los estándares de la ingeniería de software.*
- IIC2113 - Diseño detallado de software
- IIC2143 - Ingeniería de software
- IIC2154 - Proyecto de especialidad
- IIC2173 - Arquitectura de sistemas de software
- IIC2333 - Sistemas operativos y redes
- IIC2413 - Bases de datos
- IIC2713 - Sistemas de información
- IIC3113 - Gestión de proyectos de TI

#### **c. Investigación Aplicada**
*Realizar investigación aplicada a un nivel básico en tecnologías de punta existentes en la industria (bases de datos, sistemas operativos, redes, ingeniería de software, arquitectura).*
- IIC2113 - Diseño detallado de software
- IIC2143 - Ingeniería de software
- IIC2173 - Arquitectura de sistemas de software
- IIC2333 - Sistemas operativos y redes
- IIC2343 - Arquitectura de computadores
- IIC2413 - Bases de datos

#### **d. Trabajo en Equipo Multidisciplinario**
*Trabajar y colaborar con éxito en equipos, asumiendo con responsabilidad diversos roles en entornos multidisciplinarios.*
- IIC2143 - Ingeniería de software
- IIC2154 - Proyecto de especialidad
- IIC2413 - Bases de datos
- IIC2513 - Tecnologías y aplicaciones web
- IIC2713 - Sistemas de información
- IIC3113 - Gestión de proyectos de TI

#### **e. Comunicación Efectiva**
*Comunicarse en forma efectiva en diversos contextos, tanto en forma oral como escrita, y en castellano e inglés.*
- IIC2143 - Ingeniería de software
- IIC2154 - Proyecto de especialidad
- IIC3113 - Gestión de proyectos de TI
